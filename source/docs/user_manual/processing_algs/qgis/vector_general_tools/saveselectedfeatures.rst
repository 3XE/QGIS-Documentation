Save selected features
======================

Description
-----------

Saves the selected features as a new layer.

Parameters
----------

``Input layer`` [vector: any]
  Layer to process.

Outputs
-------

``Output layer with selected features`` [vector]
  The resulting layer.

Console usage
-------------

::

  processing.runalg('qgis:saveselectedfeatures', input_layer, output_layer)
