Convert polygon/line vertices to points
=======================================

Description
-----------

Converts the line or polygon vertices into points.

Parameters
----------

``Shapes`` [vector: any]
  Layer to process.

Outputs
-------

``Points`` [vector]
  The resulting layer.

Console usage
-------------

::

  processing.runalg('saga:convertpolygonlineverticestopoints', shapes, points)
